Expectations

Use Raspberry Pi Zero W and Raspberry Pi Pico to display daily quotes from Internet. With RP Zero W always plugged, the quotes can be always displayed on OLED.

Devices Used

  • Raspberry Pi Zero W
  • Raspberry Pi Pico
  • OLED display module of 0.96 inch
  • Jumper wires

Packages Used

  • urllib
  • BeautifulSoup
  • ssd1306
  • machine
  • utime
